# Eunoform Companion — Phase 1 Code Drop

This package is designed to be copied into the root of the existing `eunoform_companion` repository.

## Install and test

```bash
uv sync --dev
uv run pytest
```

Or:

```bash
python -m pip install pytest
pytest
```

See `docs/05_Phase_1_Implementation_Notes.md` for design decisions and covered acceptance behaviors.
